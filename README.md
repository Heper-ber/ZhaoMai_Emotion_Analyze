## 🌟 核心引擎逻辑 (Core Engine Logic)

### 1. 文本与色彩的底层映射 (Text-to-Color Mapping)
系统最底层的“基因”，是一套极具文学通感的“词-色”映射矩阵。每一处文字的色彩变化并非随机生成，而是基于文本语境构建的启发式情感字典。系统剥离了文字的表层语义，将其还原为 RGB 视觉信号。

### 2. 具有“记忆”的情绪动量算法 (Emotional Momentum)
文学中的情绪不是孤立的断点，而是绵延的河流。本系统的算法核心在于处理情绪状态的连续性与突变性：
* **情感惯性：** 每一次色彩的刷新都与前一刻高度相关，通过阻尼渐变保留情感的“余温”。
* **突发冲激：** 当系统探测到极具爆发力的强情绪词时，算法会触发急转弯机制，瞬间赋予当前情绪 90% 的权重，打破平滑渐变，完美还原生理性的心理震颤。

### 3. 基于荣格“无意识理论”的三级情绪网 (Jungian 3-Layer Net)
> **💡 原创声明 (Originality Note)：** 本架构为作者**独立构思与推演的原创计算模型**。在未参考工业界现有 AI 情感分析框架的前提下，作者跨界借鉴了心理学理论，旨在解决传统 NLP（自然语言处理）缺乏“文本上下文潜意识”的痛点。

系统架构借鉴了荣格（Carl Jung）的模型，构建了三层递进的递归网络：
* **大网 (Global Net) - 深层无意识：** 提取全书的核心意象，形成宿命般的基底色（30% 权重）。它是人物内心最深处的潜意识底色，如影随形。
* **中网 (Middle Net) - 前意识/情境场：** 动态侦测 10 个叙事锚点，构建当前情节的环境光晕（70% 权重）。它代表着逐渐向上层浮现的无意识冲动。
* **小网 (Small Net) - 意识神经末梢：** 捕捉当前句子的瞬时情绪，化作前端剧烈跳动的心电图锯齿。通过底层网络的“基因渗透”，意识始终被无意识所包裹与牵引。

### 4. 【核心演进】双层期望偏差机制 (Dual-Layer Expectation Deviation)
> **💡 原创声明 (Originality Note)：** 本机制为作者**纯粹基于物理距离公式与文学直觉独立推演出的“野生”计算叙事学理论**。即便其核心思想与目前前沿的动态心流控制（Dynamic Flow Control）有异曲同工之妙，但本算法运用“RGB 色彩欧几里得距离”来量化叙事张力的数学模型，系作者独立首创。

系统引入了张力模型，让代码不仅能“读懂情绪”，更能“审视剧本节奏”：
* **微观内部偏差 (Micro-Tension)：** 计算瞬时动作（小网）与当前场景氛围（中网）的 RGB 欧几里得距离。算出的 $\Delta RGB$ 张力值直接接管物理引擎，让底层波浪产生动态的“呼吸与心悸”。
* **宏观叙事偏差 (Macro-Tension)：** 独创双重引力算法。提取当前章节的中网底色，按照 **70% 剧情惯性（与上一章的色差距离） + 30% 宿命引力（与全书大网底色的距离）** 进行加权微积分。最终在界面右上角生成全书的“剧本心电图”，以上帝视角精准定位文学高潮。

## 🛠 技术实现 (Tech Stack)
* **自然语言处理：** `Jieba` 分词与自定义意象锚点识别。
* **渲染与物理引擎：** 基于 `Pygame` 引擎。引入全局时钟与 `dt`，运用线性插值 (Lerp) 实现无缝过渡，结合阻尼噪音函数构建平滑呼吸波纹。
* **架构设计 (MVC Pattern)：** 高内聚低耦合设计。彻底解耦渲染视图层、情绪特征计算层与底层语义映射层。

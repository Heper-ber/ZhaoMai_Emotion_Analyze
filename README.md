# ZhaoMai: Textual Emotion Development Projector
### (赵麦：文本情绪显影放映机)

本项目是一个跨学科的数字人文（Digital Humanities）探索项目。项目所使用的文本数据来源于我的**原创短篇小说**《最后的篝火晚会》。

与传统的关键词频率统计不同，本项目旨在打破“文学创作”与“机器感知”的壁垒，通过一套自研的情感显影算法，将小说的深层心理叙事转化为可视化的动态视觉场域。

---

## 🌟 核心引擎逻辑 (Core Engine Logic)

### 1. 文本与色彩的底层映射 (Text-to-Color Mapping)
系统最底层的“基因”，是一套极具文学通感的“词-色”映射矩阵。每一处文字的色彩变化并非随机生成，而是基于文本语境构建的启发式情感字典。系统剥离了文字的表层语义，将其还原为 RGB 视觉信号。例如，“3:46”被赋予深邃的幽暗蓝调，而“满脸冷汗”则是带着战栗感的冷灰，实现了语言向视觉的精准翻译。

### 2. 具有“记忆”的情绪动量算法 (Emotional Momentum)
文学中的情绪不是孤立的断点，而是绵延的河流。本系统的算法核心在于处理情绪状态的**连续性与突变性**：
- **情感惯性：** 每一次色彩的刷新都与前一刻的色彩高度相关，通过阻尼渐变保留情感的“余温”。
- **突发冲激：** 当系统探测到诸如“哭腔”、“攥紧”等极具爆发力的强情绪词时，算法会触发急转弯机制，瞬间赋予当前情绪 **90% 的极高权重**，打破平滑的渐变，实现画面色彩的“瞬时痉挛”，完美还原生理性的心理震颤。

### 3. 基于荣格“无意识理论”的三级情绪网 (Jungian 3-Layer Net)
如果说小说的字面意义是“意识”，那么其背后涌动的氛围则是“无意识”。系统架构借鉴了荣格（Carl Jung）的心理学模型，构建了三层递进的递归网络：
- **大网 (Global Net) - 深层无意识：** 提取全书的核心意象，形成宿命般的基底色（30% 权重）。它是人物内心最深处的潜意识底色，如影随形。
- **中网 (Middle Net) - 前意识/情境场：** 动态侦测 10 个叙事锚点，构建当前情节的环境光晕（70% 权重）。它代表着逐渐向上层浮现的无意识冲动。
- **小网 (Small Net) - 意识神经末梢：** 捕捉当前句子的瞬时情绪，化作前端剧烈跳动的心电图锯齿。
通过底层网络的“基因渗透 (Alpha Blending)”，意识（跳动的线条）始终被无意识（背景环境光）所包裹与牵引。

## 🛠 技术实现 (Tech Stack)
- **自然语言处理：** `Jieba` 分词与自定义意象锚点识别。
- **渲染与物理引擎：** `Pygame` 引擎，引入全局时钟与 `dt` (Delta Time) 解决帧率同步，运用线性插值 (`Lerp`) 实现无缝的色彩与物理振幅过渡。
- **面向对象架构：** 高内聚低耦合设计，解耦渲染层 (`main.py`)、网络架构层 (`emotion_net.py`) 与映射层 (`color_mapping.py`)。
